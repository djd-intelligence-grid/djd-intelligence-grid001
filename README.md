# Issue Compass — Republic Portal

A civic issue-education page: search by ZIP or district, see what candidates
have said, sponsored, or voted on for the issues that affect that community —
sourced directly from primary sources.

## What this is
- **Issue-first, not candidate-first.** Every claim links to a primary source
  (bill text, public statement, voting record).
- **No rankings, no endorsements.** Republic Portal does not rank, score, or
  recommend any candidate or party.
- **Community Wall** — an equal-terms, self-posted opinion section, clearly
  separated from sourced record content.

## Status
Early build. Candidate/record content currently shown is **sample/placeholder
data** pending real sourced content.

## Stack
Static HTML/CSS/JS. Part of the DJD Intelligence Grid.
